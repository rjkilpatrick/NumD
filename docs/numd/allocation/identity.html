    Alias identity

    Allocates 2D identity sliced array with ones on the major diagonal and zeros elsewhere

    alias identity = eye;

    Example

    Slice!(int*, 2) eye = ones!int(4);
Slice!(double*, 2) matrix = ones!double(4, 2);
